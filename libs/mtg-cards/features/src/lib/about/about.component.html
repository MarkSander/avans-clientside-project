Magic The Gathering

Magic The Gathering is een fysiek kaartspel waarbij je met minimaal 2 spelers speelt totdat je de laatste bent die het overleeft. Bij dit spel heb je talloze kaarten. Er zijn verschillende typen kaarten zoals creatures, landen , instants en artifacts. Daarbij kost iedere kaart ook een in game valuta genaamd "mana" om neergelegd te kunnen worden. Ieder jaar komen er minstens 4 nieuwe sets uit wat weer duizenden nieuwe kaarten toevoegd. Een kaart heeft hierbij een naam, het type kaart, de set, prijs, de zeldzaamheid van de kaart, en de manakost om de kaart neer te kunnen leggen. Naast losse Kaarten kun je van een set ook een booster kopen. Dit is een pakketje dat een verschillend aantal random kaarten bevat van de set. Hierbij heb je ook verschillenden typen boosters. Een booster bevat een naam, type, aantal kaarten, set en de prijs. Als je dat nog te weinig vindt bestaat er ook nog een boosterbox. Dit is een doos met allemaal boosters van een bepaalde set. een boosterbox heeft een naam, aantal boosters, set en een (over het algemeen dure) prijs.
